# Название проекта: CompareRepositoryTutor

## 1. Общее описание
**CompareRepositoryTutor** — это специализированное приложение, предназначенное для экспериментального сравнения алгоритмов поиска максимального общего транзитивного подграфа (MCTS) в графах версий Git. Система выступает как мощный вспомогательный инструмент в процессе обучения студентов работе с системами контроля версий, автоматизируя проверку и визуализацию графов истории коммитов.

## 2. Визуализация и сравнение
Система предоставляет интерактивный интерфейс для анализа расхождений между графами. Основные методы сравнения:
- **Two Graph Comparison**: Позволяет side-by-side отобразить граф работы студента и эталонный граф.
- **Merged Graph Analysis**: Позволяет объединить оба графа для поиска общего транзитивного подграфа, наглядно выделяя общие и уникальные части.

### Легенда статусов узлов
Для быстрого понимания результатов сравнения используется цветовая индикация узлов (коммитов):
- **IDENTICAL** (Зеленый): Узел полностью совпадает в обоих графах.
- **MODIFIED** (Желтый): Узел найден в обоих графах, но содержимое (diff-ы, метаданные) отличается.
- **EXTRA** (Красный): Узел присутствует только в одном из графов (лишний или пропущенный).
- **MOVABLE** (Синий/Фиолетовый): Узел идентичен по содержимому, но имеет иную позицию в транзитивной структуре (перемещен).

## 3. Функциональные модули

### Управление лабораторными работами и задачами
Администратор системы имеет доступ к функционалу управления учебным процессом:
- Создание, редактирование и удаление лабораторных работ.
- Создание задач для лабораторных работ с привязкой к эталонным репозиториям.
- Назначение задач конкретным студентам для выполнения.

### Студенческий интерфейс
Студенты могут:
- Просматривать список назначенных лабораторных работ и задач.
- Загружать решения в формате ZIP-архивов.
- Запускать процесс автоматической проверки решения и переходить к детальному отчету о сравнении их графа с эталонным.

## 4. Как провести сравнение
Для начала работы перейдите в раздел "Сравнение графов":
1. **Введите пути**: Укажите путь к локальному репозиторию студента и эталонному репозиторию.
2. **Выберите метод**: Выберите подходящий алгоритм (Two Graph или Merged).
3. **Запустите**: Нажмите кнопку "Сравнить". Система выполнит расчет MCTS и перенаправит вас на страницу с интерактивным отчетом.
